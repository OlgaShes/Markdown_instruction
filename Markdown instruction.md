## Списки

Чтобы добавить ненумерованные списки, необходимо начинать пункты со звездочки (*). Например, вот так:
* Элемент 1
* Элемент 2
* Элемент 3

Чтобы добавить нумерованные списки, необходимо начинать пункты с нумерации с точкой. Например, вот так:
1. Первый пункт
2. Второй пункт
